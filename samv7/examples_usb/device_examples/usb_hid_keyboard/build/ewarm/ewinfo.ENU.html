usb_hid_keyboard

The demo simulates a simple keyboard with a caps lock and 'a' on it. When a board running this program connected to a host (PC for example), with USB cable, the board appears as a HID Keyboard for the host. Then you can use the push buttons on the board to input letter to the host. E.g, to open a editor and input a letter 'a'.
